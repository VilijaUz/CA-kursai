Git ir Github

Git - kodo versijavimo programa.

GitHub - svetaine,kurioje yra keliamos kodo versijos.

  1. Isirasyti Git'a i kompiuteri
  2. Patikrinti ar git'as irasytas-terminale parasyti "git --version"
  3. Susikurti gitHub'o paskyra
  4. Konfiguruoti git'
  5. git config --global user.name gitHubVartotojoVardas
  6. git config --global user.email gitHubVartotojoEmail
  7. git config --global core.editor code
  8. git config --global core.autocrlf true (true jei Windows)
  9. Patikrinti ar teisingas config failas - git config --global --edit
