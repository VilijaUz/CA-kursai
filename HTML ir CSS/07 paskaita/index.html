Formos elementu atributai

  • for - label atributas-nurodo kuriam ivestuees laukui priskirtas label'is

  • form -formos tag'as
  • label - etikete-naudojama prie ivesties nurodyti ivesties laukelio pavadinima
  • input - ivesties laukelis skirtas DAUG KAM(priklauso nuo atributo reiksmes0
  • textarea - ivesties laukelis skirtas dideliam teksto kiekiui
  • select -ivesties laukas skirtas pasirinkti vienam is pateiktu variantu
  • option - select'o pasirinkimo variantai
  • optgruop - option'u grupes
  • output - formos isvesties vieta
  • id - *aributas -sujungti label su ivesties lauku;ivesties lauko atpazinimui per JS
  • name - ivesties lauko atributas -skirtas atpazinti ivesties lauka
  • placeholder - ivesties lauko atributas - skirtas atvaizduoti pavyzdine ivesti
  • rows/cols - textarea atributas - skirtas nurodyti textarea dydi(geriau naudoti CSS-a
  • label -optgruop atributas - skirtas nurodyti optio'u grupes pavadinimui
  • value -* atributas -nurodo ivesties reiksme
  • selected -option atributas -nurodo automatiskai pasirinkta option'a
  • type input atributas-daug opciju:
    • button - mygtukas(retai naudojamas)
    • color - spalvos pasirinkimas
    • date -
    • datetime -ir valanda pasirenkama
    • email reikalauja formato email'o
    • file - pasirinkti kompiuterio fail'a
    • hidden - paslepia input'a(neneaudojamas)
    • image - formos pateikimo mygtukas paveiksliuko pavidalu
    • month - metu/menesio pasirinkimas
    • number - min/max/zingsnis gb nurodyma
    • password - slaptazodis,kuri maskuoja akis
    • range - slankiklis -is bibliotekos pasiimti -nuo..iki
    • reset - isvalo forma
    • search - paieskai skirtas laukas
    • submit - mygtukas formos pateikimui
    • tel - telefono -pattern-nusako kokiu formatu turi buti uzrasytas tekstas
    • text -tekstas
    • time - tik valandos
    • url - nuorodos
    • week - pasirinkti savaite
    • checkbox
    • radio - vienas pasirinkimas
    • checked -radio/checkbox pradinis pazymetas radio box'as
