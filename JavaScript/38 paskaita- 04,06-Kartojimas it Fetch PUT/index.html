Kintamieji

kintamieji skirti saugoti/laikyti ivairia informacija programos veikimo metu:

  • string -tekstas
  • number -skaicius
  • boolen - loginis
  • array -masyvas
  • object

Aritmetika

Kazkokie veiksmai atliekami tarp duomenu/kintamuju.

  • +
  • -
  • * - daugyba
  • / - dalyba
  • ** - kelimas laipsniu
  • **(1/y) - saknies traukimas
  • % - liekanos gavimas

Palyginimai

Palygini keleta duomenu.Ir palyginimas grazina true arba false.(truthy ir falsy)

  • > - daugiau
  • < - maziau
  • >=
  • <=< /li>
  • == lygu(reiksmes)
  • === lygu(reiksmes ir tipai)
  • != - nelygu(reikmes)
  • !== - nelygu(reiksmes ir tipai)
  • ! -(NOT)- apvercia boolean reiksme
  • && -(AND)- tikrina ar abiejose pusese salyga,grazina true
  • || -(OR) -tikrina ar bent vienoje

Funkcijos

Funkcijose esamas kodas vukdomas kodas vykdosi TIK tuomet,kai funkcija yra iskvieciama.
Funkcija gali buti iskvieciama daugeli kartu.
Funkcija priima parametrus(duomenis) ,su kuriais atlieka kazkokius veiksmus.Kreipiantis i funkcija yra perduodami argumentai(duomenys). Funcijos viduje jie vadinami parametrais ,o ne argumentais. (perpanaudojamas kodas) Funkcija gali grazinti kazkoki rezultata. Funkcijos gali buti perduotos kaip parametrai ir tuomet iskvieciamos; tai vadinama callback. br> callback'ai dazniausiai aprasomi kaip bevardes funkcijos. Tokiu atveju ju nereikia perduoti kaip parametru.

  • function name()- paprasta f.
  • const name = () => {} - arrow/lamda f.
  • function(){} - bevarde f.
  • ()=>{} - bevarde arrow f.(bevarde f. rasoma callback viduje)

NODE

Selektinimas

Is JS kreipiamai i HTML ir selektinama taip pat kaip ir CSS'e kazkoks arba kazkokie elementai.

  • querySelector('selektorius')-grazina rasta elemnta kaip Node'a
  • querySelectorAll('selektorius')-grazina visus rastus elemntus kaip NodeList'a (beveik tas pats kaip masyvas)
  • Senieji:
    • getElementById('id')
    • getElementByClassName('class')
    • getElementByTagName('tagName')

Formu nuskaitymas

Forma yra nuskaitoma po to ,kai ivyksta formos submit eventa'as. Per JS'a sulauke formos pateikimo,sustabdome puslapio persikrovima su event.preventDefault() ir susirenkame formos ivesciu laukuose pateikta informacija.
Pateikta informacija susirenkame kreipdamiesi i formos elementu vardus/id.

Metodai

Metodas yra funkcija,priskirta scecifiniam objektui/masyvui

Masyvu metodai

Funkcijos skirtos tik masyvams.Ju yra labai daug ir iv formu.
Masyvai turi iteracinius metodus.Jie skirti sukti cikla per veiksmus su kiekvienu masyvo elementu.

Syntax

masyvas.iteracinisMetodas((masyvoElementas,iteracija,visasMasyvas)=>{veiksmai...})

Metodu ivairove

  • String
  • Number
  • Math
  • Date
  • Array
  • Object
  • JSON
  • Global_Object....

Destrukturizavimas

Isardyti objekta/masyva/gaunamusParametrus.... i mazesnes dalis.

I funkcija/metoda/klase gaunamas vienas DIDELIS parametras ir is jo galima pasiimti tik reikalingas reiksmes. Galima spread'inti masyva i atskirus gabaliukus.

MODULIAI

Vieno failo duomenu(kintamieji,klases,funkcijos...) perdavimas i kitus failus. Export ir Import

Klases

Klases yra skirtos objektams.
Klasei galima perduoti argumentus,kuriuos klase konstruktoriaus viduje priima ir pasivadina kazkokiais pavadinimais,kuriuos naudoja visos klases viduje.

Pagrindiniai klases principai

  • Inkapsuliacija - viskas ko klasei reikia yra jos viduje
  • Abstrakcija - is isores negalima pasiekti,supaprastina kuriamo objekto naudojima
  • Paveldimumas -kuriant klases,galima pavedeti kitas klases
  • Polimorfizmas - tos pacios klases objektai gali tureti skirtingas strukturas
