Moduliai

Moduliai(moduliarizacija)-kazkokiu funciju/kintamuju/klasiu ... laikymas atskiruose failuose ir ju export'avimas ir import'avimas
Moduliai naudojami tam,kad visoko nelaikyti viename faile,kas projektui pleciantis gali tapti labai sudetinga susigaudyti faile.

Export - suteikiama galimybe kazkokioms funkcijoms/kintamujiesiems/klasems ... buti pasiekiamiems is kitu failu.
export default name - eksportuojamas pagrindinis failo elementas. Gali buti tik vienas defaultinis export'as per viena faila. export{name1,name2,....}-eksportuojami papildomi elementai.Galima eksportuoti kiek nori,juos rasant i {} ir atskiriami kableliais.

Import-suteikiama galimybe pasiekti kazkokias funkcijas/kintamuosius/klases ... is kitu failu.
import anyNameYouWant from './fileRouteAndFile'- importuojamas pagrindinis nurodyto failo elementas.Ji galime importuoti kokiu tik patys norime vardu. import {name1,name2,....} from'./fileRouteAndFile'-importuojami TIK nurodyti papildomi elementai. Importuojant papildomus elementus,reikia nurodyti ju SPECIFINIUS vardus.
import{ name1 as simethingElse,name2,...} from'./fileRouteAndFile'- po importuojamo importuojamo elemento pavadinimo parasius zodeli "as",galima nurodyti kokiu kitu pavadinimu i ta elementa siame faile kreiptis.
import anyNameYouWant,{name1 as somethingElse,name2,.. } from'./fileRouteAndFile - galima importuoti IR default,IR paprasta,IR paprasta pervadinta.
