Komandines eilutes komandos

  • TAB - uzbaigia rasoma eilute arba pasiulo variantus( nenaudoti ant tuscios komandine eilutes)
  • ls -listina direktorijas ir failus
  • cd-direktorijos keitimas(change directory)
  • CTRL C -atsaukia vykdoma komanda/stabdo procesus
  • clear - isvalo terminalo eilutes
  • direktorijos kurimas,failu kurimas,trynimas,perkelimas/pervadinimas,failu koregavimas
  • pwd - komanda dar parodo kuriam folderi esama

Git konfiguracijos

  • git init - sukuria Git'o repozitorija
  • git add nurodytiFailai - prideti koreguotus/kurtus/naujus failus/aplankus prie paruostu failu kodu versijos kurimui(commit/ui)
  • git commit -m - is paruostu(staged) failu/aplankalu sukuriama kodo versija su atitinkama zinute (message(-m)),nusakancia kas toje kodo versijoje pakito
  • git status - grazina informacija apie dabartines kodo versijos bukle (statusa)
  • git push - nusiuncia commit'us i isorine repozitorija ( gitHub'a)
  • git pull - atsiuncia commit'us is isorines repozitorijos (gitHub'o)
